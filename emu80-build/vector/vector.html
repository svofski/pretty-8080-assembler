Вектор-06ц. Краткая справка

Общая информация

После выбора данной конфигурации из ПЗУ запускается программа-загрузчик. Дальнейшие действия с ней описаны ниже.

Загрузка файлов с помощью встроенного загрузчика

Двоичные файлы Вектора (.rom, .r0m, .vec), а также файлы Бейсика 2.5 (.cas, .bas) можно загружать с помощью пункта меню «Файл -> Загрузить и выполнить», комбинации клавиш Alt-F3 (Meta-F3), либо соответствующей кнопки на панели инструментов. Можно также перетащить нужный файл в окно эмулятора.

Запуск встроенного в загрузчик Бейсика

Для запуска встроенного Бейсика нужно нажать и удерживать F3, после чего выполнить сьрос Вектора клавишей F11. На экране будет отображаться процесс загрузки. Для запуска программы после загрузки нужно нажать F12.

Запуск встроенного в загрузчик Монитора-отладчика

Запуск встроенного Монитора производится аналогично описанному выше запуску Бейсика, но нажать и удерживать необходимо клавишу F4.

Загрузка с использованием перехватов магнитофонных операций

В эмуляторе реализован перехват операций с магнитофоном для Бейсика 2.5, Монитора и эмулятора РК/Микроши. Для загрузки файлов Бейсика (.cas) нужно воспользоваться командой «CLOAD""».

Работа с образами дисков

Файлы образов дискет (.fdd) и электронного диска (*.edd) подключаются с помощью горячих клавиш Alt-A (Meta-A) для диска «A», Alt-B (Meta-B) для диска «B» и Alt-E (Meta-E) - для электронного диска. Можно также использовать соответствующие пункты меню или кнопки на панели инструментов.

Для загрузки с электронного диска нужно перезагрузить Вектор с помощью клавиши F11. Для загрузки с диска Aнужно при этом нажать и удерживать одновременно клавиши F1+F2 (F1+F2 можно не нажимать, если электронный диск не подключен). После перезагрузки нужно нажать F12 для загрузки с диска.

Соответствие клавиш и раскладки клавиатуры

Для Вектора рекомендуется использовать «Умную» раскладку клавиатуры, ввод буквенно-цифровых символов при этом производится как обычно, раскладка переключается средствами ОС, в которой запущен эмулятор. Раскладки клавиатуры приведены ниже:
