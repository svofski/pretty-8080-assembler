Электроника КР-04. Краткая справка

Общие сведения

При старте ПК «Электроника КР-04» предлагается выбрать один из 4 режимов работы — 48K/40, 48K/80, 16K/64, 32K/64. Первые два используют цветной графический режим и совместимы с РК-86 только на уровне вызовов Монитора, вторые два совместимы с РК-86 в конфигурациях с соответственно 16 и 32 КБайт памяти. Программы, написанные специально для КР-04, самостоятельно переключают режим либо работают в любом режиме. Однако при загрузке программ для РК-86 следует при необходимости предварительно выбирать при запуске необходимый режим. Для «родных» файлов КР-04 предлагается использовать расширение «.rk4».

Загрузка с использованием перехватов магнитофонных операций

В ответ на приглашение Монитора «-->» нужно набрать «I<Enter>» и выбрать файл. В случае успешной загрузки Монитор отображает столбец из 3 цифр - начального адреса, конечного и контрольной суммы. После загрузки запуск обычно осуществляется с помощью директивы G Монитора («G<Enter>»). В случае запуска не с нулевого адреса в директиве G можно указать адрес запуска, например: «G100<Enter>». Этим способом можно также загружать .wav и .csw-файлы.

Загрузка файлов с помощью встроенного загрузчика

Двоичные файлы образов ленты «.rkr, .rk4, .rk» можно загружать с помощью пункта меню «Файл -> Загрузить и выполнить», комбинации клавиш Alt-F3 (Meta-F3), либо соответствующей кнопки на панели инструментов. Можно также просто перетащить нужный файл в окно эмулятора. При этом в текущей версии эмулятора всегда используется режим 32K/64, так как именно в этом режиме работают большинство файлов для РК-86. В случае необходимости загружать файлы РК-86 в другом режиме, следует предварительно выбрать режим и использоватьдирективу «I» (см. предыдущий пункт).

Работа с SD-контроллером (по схеме vinxru)

В конфигурации с SD-контроллером эмулируется SD-карта, содержащая некоторые программы, разработанных для этого ПК. Для загрузки оболочки необходимо выполнить директивы Монитора «R,FF<Enter>G<Enter>». Далее выбор и запуск файла осуществляется с помощью клавиш управления курсором. Для включения или выключения поддержки SD-контроллера, а также для выбора директории SD-карты используется комбинация клавиш Alt-F8 (Meta-F8) или соответствующая кнопка на панели инструментов.

Соответствие клавиш и раскладки клавиатуры

По умолчанию для КР-04 установлена «умная» раскладка клавиатуры, ввод буквенно-цифровых символов в большинстве приложений при этом производится как обычно, раскладка переключается средствами ОС, в которой запущен эмулятор. Для работы с приложениями, использующими строчные буквы (например, текстовым редактором, идущим в комплекте), лучше использовать QWERTY и ЙЦУКЕН-раскладки.
