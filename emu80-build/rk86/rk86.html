Радио-86РК. Краткая справка

Загрузка файлов с помощью встроенного загрузчика

Двоичные файлы образов ленты «.rkr, .gam, .rk» можно загружать с помощью пункта меню «Файл -> Загрузить и выполнить», комбинации клавиш Alt-F3 (Meta-F3), либо соответствующей кнопки на панели инструментов. Можно также просто перетащить нужный файл в окно эмулятора.

Загрузка с использованием перехватов магнитофонных операций

В ответ на приглашение Монитора «-->» нужно набрать «I<Enter>» и выбрать файл. В случае успешной загрузки Монитор отображает столбец из 3 цифр - начального адреса, конечного и контрольной суммы. После загрузки запуск обычно осуществляется с помощью директивы G Монитора («G<Enter>»). В случае запуска не с нулевого адреса в директиве G можно указать адрес запуска, например: «G100<Enter>». Этим способом можно также загружать .wav и .csw-файлы.

Работа с образами дисков

Файлы образов дисков (.rkd, .rkdisk) подключаются с помощью горячих клавиш Alt-A (Meta-A) для диска «A» и Alt-B (Meta-B) для диска «B». Можно также использовать соответствующие пункты меню или кнопки на панели инструментов. Для загрузки RK-DOS нужно набрать «GE000» (универсальный способ) либо «U» (работает в версии Монитора, поставляемой с эмулятором). Для отображения перечня файлов на диске служит команда «DIR», для запуска файла нужно набрать его имя.

Работа с SD-контроллером (по схеме vinxru)

В конфигурации с SD-контроллером эмулируется SD-карта, содержащая большую часть программ, разработанных для этого ПК. Для загрузки оболочки необходимо выполнить директивы Монитора «R,FF<Enter>G<Enter>». Далее выбор и запуск файла осуществляется с помощью клавиш управления курсором. Для выбора директории SD-карты используется комбинация клавиш Alt-F8 (Meta-F8) или соответствующая кнопка на панели инструментов.

Работа с SDOS и SD-адаптером (по схемам HWM/PVV, MSX, n8vem)

В данной конфигурации эмулируется SD-адаптер по схемам HWM/PVV, n8vem (прямое подключение либо на ВВ55) и MSX. Вместе с эмулятором поставляется образ SD-карты sd_rk86.img, содержащий большую часть программ, разработанных для РК-86. Для запуска SDOS оболочки необходимо набрать «GE000» (универсальный способ) либо «U» (работает в версии Монитора, поставляемой с эмулятором). Просмотр содержимого диска - команда «DIR», переход в директорию — «CD <имя>», запуск — «<имя>». Для выбора другого образа SD-карты и изменения типа эмулируемого адаптера используется комбинация клавиш Alt-F8 (Meta-F8) или соответствующая кнопка на панели инструментов. Для SDL/wx- версии тип адаптера и имя файла образа меняются в конфигурационном файле.

Конфигурация с ROM-диском

Для работы с ROM-диском в текущей версии эмулятора необходимо раскомментировать "define ROM_DISK" в конфигурационном файле rk86.conf. Таймер в этом режиме недоступен. Для запуска оболочки необходимо набрать следующие директивы Монитора: «R7E00,7FFF,7400<Enter> G7400<Enter>». Для выбора файла образа ROM-диска используется комбинация клавиш Alt-F8 (Meta-F8) или соответствующая кнопка на панели инструментов.

Соответствие клавиш и раскладки клавиатуры

По умолчанию для РК-86 установлена «Умная» раскладка клавиатуры, ввод буквенно-цифровых символов при этом производится как обычно, раскладка переключается средствами ОС, в которой запущен эмулятор.
