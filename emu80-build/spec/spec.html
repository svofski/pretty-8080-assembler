Специалист. Краткая справка

Общая информация

После запуска платформы в ответ на запрос «PROGRAM?» следует нажать на любую из клавиш F1-F11 (появится мигающий курсор) и затем Enter для выхода в Монитор.

Загрузка файлов с помощью встроенного загрузчика

Двоичные файлы образов ленты «.rks» можно загружать с помощью пункта меню «Файл -> Загрузить и выполнить», комбинации клавиш Alt-F3 (Meta-F3), либо соответствующей кнопки на панели инструментов. Можно также просто перетащить нужный файл в окно эмулятора.

Загрузка с использованием перехватов магнитофонных операций

В ответ на приглашение «* МОНИТОР ?» нужно набрать «R<Enter>» и выбрать файл. В случае успешной загрузки Монитор отображает контрольную сумму, начальный и конечный адреса. После загрузки запуск обычно осуществляется с помощью директивы G Монитора («G<Enter>»). В случае запуска не с нулевого адреса в директиве G можно указать адрес запуска, например: «G100<Enter>». Этим способом можно также загружать .wav и .csw-файлы.

Соответствие клавиш и раскладки клавиатуры

По умолчанию для Специалиста установлена Qwerty-раскладка клавиатуры, с однозначным соответствием клавиш Специалиста клавишам PC. «Умная» smart-раскладка на данный момент не поддерживается.
