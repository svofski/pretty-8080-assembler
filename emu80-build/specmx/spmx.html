Специалист-MX. Краткая справка

Общая информация

После запуска данной платформы с ROM-диска загружается оболочка RAMFOS. Вначале выдается запрос текущей даты, можно ввести дату либо просто нажать Enter. После этого на экране отображается содержимое RAM-диска, обычно пустое сразу после запуска.

Загрузка файлов с помощью встроенного загрузчика

Файлы .cpu/.i80 можно загружать и запускать с помощью встроенного в эмулятор загрузчика. Для этого следует использовать пункт меню «Файл->Загрузить и выполнить», комбинацию клавиш Alt-F3 (Meta-F3), либо соответствующую кнопку на панели инструментов. Можно также перетащить нужный файл в окно эмулятора. Можно также просто загрузить файл на RAM-диск B без запуска. Для этого следует использовать пункт меню «Загрузить», комбинацию клавиш Alt-L, либо кнопку на панели инструментов. Исполняемые файлы имеют расширение EXE, для последующего запуска с RAM-диска нужно нажать на файле Enter.

Работа с образами дисков

Для работы с дискетой необходимо предварительно подключить образ диска (*.ord) с помощью горячих клавиш Alt-A (Meta-A) для диска «A» и Alt-B (Meta-B) для диска «B». После этого следует нажать F6 для загрузки с диска. В случае формата диска RAMFOS отображается его содержимое в левой части и содержимое RAM-диска - в правой. Для запуска файла служит клавиша F6, для копирования его на RAM-диск - клавиша F1. В случае, если дискета содержит ОС CP/M, загрузится данная ОС, работа с ней производится как обычно. На некоторых дискетах может запуститься файловый менеджер Bridge Panels.

Соответствие клавиш и раскладки клавиатуры

Для Специалиста-MX рекомендуется использовать «Умную» раскладку клавиатуры, ввод буквенно-цифровых символов при этом производится как обычно, раскладка переключается средствами ОС, в которой запущен эмулятор.

В случае запуска в Специалисте-MX некоторых программ, рассчитанных на «обычный» Специалист, а также при запуске конфигураций «MXOS» или «Commander», возможно, потребуется переключение в настройках типа клавиатуры с «RAMFOS» на «Оригинальную». Раскладки оригинальной клавиатуры приведены в соответствующем разделе справки.
