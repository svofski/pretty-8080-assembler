Микро-80. Краткая справка

Загрузка файлов с помощью встроенного загрузчика

Двоичные файлы образов ленты «.rk8» можно загружать с помощью пункта меню «Файл -> Загрузить и выполнить», комбинации клавиш Alt-F3 (Meta-F3), либо соответствующей кнопки на панели инструментов. Можно также просто перетащить нужный файл в окно эмулятора.

Загрузка с использованием перехватом магнитофонных операций

В ответ на приглашение Монитора «>» либо «-->» нужно набрать «I<Enter>» и выбрать файл. В случае успешной загрузки Монитор отображает столбец из 3 цифр - начального адреса, конечного и контрольной суммы. После загрузки запуск обычно осуществляется с помощью директивы G Монитора («G<Enter>»). В случае запуска не с нулевого адреса в директиве G можно указать адрес запуска, например: «G100<Enter>». Этим способом можно также загружать .wav и .csw-файлы.

Работа с ROM-диском

Для запуска оболочки ROM-диска необходимо воспользоваться конфигурацией с Монитором РК и набрать следующие директивы Монитора: «R7E00,7FFF<Enter> G<Enter>» (оболочка перемещаема и может быть запущена с любого адреса). Для выбора файла образа ROM-диска используется комбинация клавиш Alt-F8 (Meta-F8) или соответствующая кнопка на панели инструментов. Поставляемый с эмулятором образ ROM-диска взят из проекта https://github.com/prokushev/micro-80-cpm и содержит 2 версии интерпретатора языка Basic, игру Тетрис и ОС CP/M, рассчитанную на работу с электронным диском.
